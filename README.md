# Fashion KAN Recommendation System

A fashion recommendation system built using Kolmogorov-Arnold Network (KAN) and H&M fashion dataset.

## Project Overview

This project recommends fashion products to users based on user age, product features, price and category information.

## Technologies Used

- Python
- PyTorch
- KAN (Kolmogorov-Arnold Network)
- Pandas
- Scikit-learn
- Streamlit

## Dataset

H&M Fashion Dataset containing:

- Customer information
- Article information
- Transaction history

## Machine Learning Pipeline

Dataset
↓
Data Preprocessing
↓
Feature Encoding
↓
Negative Sampling
↓
KAN Model Training
↓
Evaluation
↓
Product Recommendation

## Model Features

Input Features:

- Age
- Price
- Product Type
- Colour
- Department

## Evaluation Metrics

The model is evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score

## Results

Accuracy: 51.90%

Precision: 0.5032

Recall: 0.7353

F1 Score: 0.5975
